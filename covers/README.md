# Book covers

Drop book cover images here and reference them from `art/manifest.js`
(`cover: "covers/<file>.jpg"`).

The Spiroglyphics covers are square — tiles are square to match, shown in
the "Artworks" switcher (40×40) and on the completion book card (56×56).
JPG or PNG, roughly 300×300 px is plenty.

Until an image file exists, the page shows a lettered tile automatically,
so missing covers never break anything.
